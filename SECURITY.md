# Security Policy

## Supported Versions

This project is currently in early development. Security fixes are handled on the `main` branch.

## Reporting a Vulnerability

Please report security issues privately to ASTUS LAB WEB STUDIO before opening a public issue.

Website: https://astuslab.com.ua/

Include:

- affected version or commit;
- device and Android version;
- steps to reproduce;
- expected and actual behavior;
- logs or screenshots when relevant.
