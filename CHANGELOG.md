# Changelog

## 0.1.0

- Initial Android app project.
- Added library, import, reader, TTS, settings, and manual screens.
- Added Room persistence for books, bookmarks, reading progress, and TTS progress.
- Added DataStore settings for TTS speed, pitch, engine, theme, and screen-on behavior.
- Added support for TXT, FB2, FB2.ZIP, EPUB, PDF, DOCX, HTML, Markdown, and RTF import.
- Added foreground TTS playback service.
- Added Android App Widget.
- Added splash artwork and launcher icon.
